CUDA

We also support CUDA.jl.

Load

Before using a CuArray simply invoke:

using CUDA

Issues with Regularizers

However, our approach to express the regularizers with Tullio.jl is currently not performant with GPUs. Therefore, to use CuArrays with regularizers, you need to choose TV_cuda.
